# Hệ thống quản lý sinh viên (QLSinhVien)

## 📌 Giới thiệu
Đây là hệ thống cơ sở dữ liệu quản lý sinh viên được xây dựng trên SQL Server.

## ⚙️ Chức năng chính
- Quản lý sinh viên
- Quản lý lớp, khoa, ngành
- Quản lý giảng viên
- Quản lý môn học và học phần
- Đăng ký học phần
- Quản lý điểm
- Thống kê và cảnh báo học vụ
